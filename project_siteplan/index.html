Project Site Plan: Light of the Gospel

Site Name

Light of the Gospel

This name was chosen because it clearly represents the purpose of the site: to share the light and principles of the Gospel of Jesus Christ.

Site Purpose

The purpose of this site is to create an online space where the Gospel of Jesus Christ is shared through personal experiences, personal study, scriptures, personal testimonies, and daily inspiration.

Scenarios

  • Where can I find teachings about the doctrines of Jesus Christ?
  • What is an inspiring Gospel quote for today?

Color Scheme

  • #022c5d — Used for headings, header and footer background, and navigation links.
  • #b9e8dc — Used as the background for highlighted blocks (such as inspiring quotes).
  • #f4f9f9 — Used for page backgrounds and light content areas.
  • #ff6b6b — Used for buttons, links, and attention-grabbing elements.

Typography

  • Roboto — Used for all general site content, including headings, paragraphs, and menus.
